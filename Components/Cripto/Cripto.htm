Descrição do Componente:

TCriptografa é um componente não visual para criptografação de textos.
Para se criptografar basta colocar uma string na propriedade Text e imediatamente poderá se verificar o resultadado nas propriedades CriptoBin e CriptoHex. O inverso também é possível: coloca-se um texto criptografado binário na propriedade CriptoBin ou um texto criptografado hexadecimal na propriedade CriptoHex e tem-se a string original na propriedade Text.

 

Propriedades:

CriptoBin: string

Esta propriedade contém a criptografação da string informada na propriedade Text. O código ASCII dos caracteres resultantes da criptografação estará entre 0 e 255. Não é ideal que se use esta propriedade para armazenamento em banco de dados e tabelas, uma vez que o driver utilizado pode armazenar de forma incorreta os caracteres cujo ASCII são superiores a 128. Quando você quizer armazenar o resultado criptografado de uma string em BD ou tabelas use o resultado contido na propriedade CriptoHex. Você deve usar a propriedade CriptoBin para arquivos texto.

Exemplo:

procedure Form1.Button1Click(Sender: TObject);
var
  C: string;
begin
  Criptografa1.Text := 'Texto normal';
  C := Criptografa1.CriptoBin;  {texto criptografado}
end;

CriptoHex: string

Contém a criptografação da string setada na propriedade Text. Os caracteres resultantes da criptografação estarão entre os caracteres '0123456789ABCDEF'. Desta forma torna-se prático o armazenamento em tabelas, porém o tamanho da string contida em CriptoHex e 2 vezes o tamanho da string em Text.

Key: string

Através desta propriedade você especificará uma string que servirá de chave e será usada no momento da criptografação. Você deve personalizar esta chave para cada sistema, assim ficará mais dificíl uma tentativa de quebra do sigilo, impedindo que outra pessoa que possua o TCriptografa consiga descriptografar suas informações.

Exemplo:

procedure Form1.Button1Click(Sender: TObject);
var
  C: string;
begin
  Criptografa1.Key := 'Tjk823';  {personaliza a chave}
  Criptografa1.Text := 'Texto normal';
  C := Criptografa1.CriptoHex;  {texto criptografado}
end;

Text: string

Contém o texto original da criptografação.

Exemplo:

procedure Form1.Button1Click(Sender: TObject);
var
  S: string;
begin
  S := Table1.FieldByName('Senha').AsString;  {pega senha criptografada}
  Criptografa1.CriptoHex := S;
  if Criptografa1.Text <> 'senha_do_sistema' then
    Close;
end;

 

Métodos:

TextToCriptoBin(Text: string): string

Este método retorna uma string criptografada de forma binária. Ou seja, o código dos caracteres que comporão a string de retorno estará entre 0 e 255.

Exemplo:

procedure Form1.Button1Click(Sender: TObject);
var
  C: string;
begin
  {a variável C conterá a criptografação binária}
  C := Criptografa1.TextToCriptoBin('algum texto');
end;

TextToCriptoHex(Text: string): string

Converte uma string para outra string criptografada no formato hexadecimal. Neste método os caracteres que comporão a string de retorno estarão entre os seguintes: '0123456789ABCDEF'.

Obs.: Neste caso o tamanho da string de retorno será o dobro do tamanho da string original. Este formato é útil para o armazenamento em tabelas.

CriptoBinToText(Text: string): string

Retorna o texto original de uma string criptografada no formato binário.

CriptoHexToText(Text: string): string

Retorna o texto original de uma string criptografada no formato hexadecimal.

Exemplo:

procedure Form1.Button1Click(Sender: TObject);
var
  S: string;
begin
  S := Table1.FieldByName('Senha').AsString;  {pega senha criptografada}
  if Criptografa1.CriptoHexToText(S) <> 'senha_do_sistema' then
    Close;
end;
