Instalação do Componente no Delphi 3

Estando com o Delphi 3 aberto siga os seguintes passos:

1. Escolha, no menu principal, a opção "Component / Install Component..."
2. Clique na segunda aba (página) "Into New Package"
3. Estando já na aba (página) "Into New Package" clique no primeiro botão "Browse" e procure pelo seguinte arquivo "nome_do_componente.dcu"
4. No campo "Package file name" entre com o seguinte texto (sem aspas): "Componentes"
5. No campo seguinte "Package description" entre com o texto (sem aspas): "TCF Componentes"
6. Em seguida clique no botão "Ok"
7. Caso durante a instalação seja pedido para adicionar alguns arquivos como "VCL30", "VCLDB30", você deve confirmar clicando no botão "Ok".

Será criada um paleta chamada TCF e nela será colocado o componente. Para visualizar esta nova paleta, role as já existentes para a esquerda.

Atenção: se ocorrer alguma mensagem solicitando o arquivo "nome_do_componente.pas" é porque você está tentando instalar um componente para uma versão do Delphi diferente da sua.
